## 测试方法

### 二分查找算法
```
1. 设定最小并发数（min）和最大并发数（max）
2. 测试中间值 mid = (min + max) / 2
3. 如果成功率 >= 95%:
   - 记录当前并发数为有效值
   - 继续测试更高并发: min = mid + 1
4. 如果成功率 < 95%:
   - 降低并发数: max = mid - 1
5. 重复步骤2-4，直到找到最大有效并发数
```

### 成功率阈值
- 设定为 95%
- 确保系统在高并发下仍能保持稳定
