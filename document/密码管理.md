## 用户前端

用户前端 (http://localhost:3001) 需要用户自行注册账号，没有默认账号。

### 注册新用户
1. 访问 http://localhost:3001
2. 点击"注册"按钮
3. 填写用户名、密码、邮箱等信息
4. 完成注册后即可登录使用

## 安全提示

⚠️ **重要安全提示**:
- 默认密码 `admin123` 仅用于开发测试环境
- 生产环境部署前，务必修改默认密码
- 建议使用强密码（至少8位，包含大小写字母、数字和特殊字符）
- 定期更换管理员密码
- 不要在公网环境使用默认密码

## 密码加密说明

系统使用 BCrypt 算法加密密码，具有以下特点：
- 每次加密同一密码会生成不同的哈希值
- 验证时会自动匹配正确的密码
- 安全性高，难以破解
- 自动加盐，无需额外配置

## 忘记密码怎么办？

如果忘记管理员密码，可以通过以下方式重置：

### 方式1: 数据库直接更新
```sql
-- 连接到MySQL
docker exec -it speakmaster-mysql mysql -uroot -proot

USE speakmaster_user;

-- 重置为 admin123
UPDATE `user` 
SET `password` = '$2a$10$vH5jYnsVAM4LyjRjrUyyEOH0yCDIhivhye82P.Qmi6N0nHym6Mh26'
WHERE `username` = 'admin';
```

### 方式2: 使用密码生成工具
运行项目中的密码生成工具：
```bash
cd backend/user_service
# 使用 Maven 运行测试类
mvn test-compile exec:java -Dexec.mainClass="com.speakmaster.user.PasswordEncoderTest"
```

## 其他账号

目前系统只有一个默认管理员账号。如需创建更多管理员或测试账号，可以：

1. 通过管理后台的"用户管理"功能创建
2. 通过用户前端注册功能创建普通用户
3. 直接在数据库中插入（需要使用 BCrypt 加密密码）
